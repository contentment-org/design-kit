# Themes

This are just static objects which are exported internally for the library.

## Supported themes

### `default`

```js
{
  // font family
  componentFontFamily: 'Varela Regular, sans-serif',
  labelFontFamily: 'Open Sans, sans-serif',

  // font sizes
  fontSizeRegular: '15px',
  fontSizeSmall: '13px',
  fontSizeTitle: '24px',
  fontSizeSmallTitle: '18px',

  // font weights
  fontWeightRegular: 400,

  // colors
  primaryColor: '#54bf97',
  secondaryColor: '#0190be',
  whiteColor: '#fff',

  // border radius
  cardBorderRadius: '3px',
  buttonBorderRadius: '4px',

  // menu level
  zIndexMenuLevel: 100,
}
```
